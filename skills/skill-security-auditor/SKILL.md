# Skill Security Auditor

基于提示词分析的技能安全审计工具，用于评估DeepSeek技能的安全性。

## 概述

Skill Security Auditor 是一个基于提示词模板的安全审计工具，专门用于分析DeepSeek技能的安全性。与传统的Python脚本审计不同，本工具使用智能提示词模板来读取和分析技能文件内容，识别潜在的安全风险。

## 核心功能

1. **技能文件分析** - 自动读取技能目录下的所有文件
2. **提示词模板审计** - 使用预定义的提示词模板分析代码安全性
3. **安全风险识别** - 检测API密钥泄露、恶意命令执行、文件系统权限等问题
4. **依赖包安全** - 分析requirements.txt和package.json中的依赖安全
5. **网络访问权限** - 检查网络请求和外部API调用
6. **生成审计报告** - 输出详细的安全审计报告

## 使用方法

### 基本使用
```
分析技能安全性：请使用Skill Security Auditor分析指定技能的安全性。
```

### 详细分析
```
对技能进行深度安全审计：请使用Skill Security Auditor对[技能名称]进行全面的安全审计，包括代码分析、依赖检查和安全风险评估。
```

### 生成报告
```
生成安全审计报告：请使用Skill Security Auditor分析[技能名称]并生成详细的安全审计报告，包含风险等级和建议修复措施。
```

## 审计流程

1. **技能文件扫描** - 读取技能目录下的所有文件
2. **内容分析** - 使用提示词模板分析文件内容
3. **安全规则匹配** - 应用安全规则模板检测风险
4. **风险评估** - 评估风险等级和影响
5. **报告生成** - 输出结构化审计报告

## 安全规则模板

工具包含以下安全规则提示词模板：

1. **API密钥泄露检测** - 检测硬编码的API密钥、令牌和密码
2. **命令执行风险** - 识别危险的系统命令执行
3. **文件系统权限** - 检查不安全的文件操作
4. **网络访问控制** - 分析外部API调用和网络请求
5. **依赖安全分析** - 检查依赖包的安全漏洞
6. **权限提升风险** - 识别可能的权限提升漏洞

## 输出格式

审计报告包含以下部分：
- 技能基本信息
- 发现的安全问题
- 风险等级评估（高/中/低）
- 详细的问题描述
- 修复建议
- 总体安全评分

查看 `examples/` 目录中的示例分析报告，了解工具的输出格式和分析深度。

## 注意事项

1. 本工具基于提示词分析，需要结合人工审查
2. 审计结果仅供参考，不保证100%准确
3. 建议定期更新提示词模板以适应新的安全威胁
4. 对于关键技能，建议进行多轮审计和人工验证
